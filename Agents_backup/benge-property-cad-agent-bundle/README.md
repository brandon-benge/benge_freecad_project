# Benge Property CAD Agent Bundle

Copy the contents into the root of `benge-property-cad`.

Included:

- `AGENTS.md`
- `.opencode/agents/*.md`
- `.agents/skills/*/SKILL.md`

The separately generated `opencode.jsonc` belongs in the repository root.

OpenCode officially supports project agent definitions under `.opencode/agents`,
project skills under `.agents/skills` or `.opencode/skills`, and project rules
in root `AGENTS.md`.
